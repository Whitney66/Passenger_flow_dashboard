昨日/今日销售业绩

数据定义与计算规则

  • 含免值品/不含免值品：依据商品大类【26】免值品进行区分。
  • 成交金额：小票表成交金额，剔除退货。对比变化率使用红色表示增加、绿色表示减少。
  • 环比 =（当日成交金额 / 昨天成交金额）* 100%。
  • 同比 =（当日成交金额 / 去年同期成交金额）* 100%。
  • 无法对比时，环比/同比展示100%。
  • 团队目标达成：通过进度条百分比表示，四舍五入取整；团队成交金额 / 目标销售额（元）。
  • 月目标：昨日/今日取自《员工销售目标维护》的【目标销售额（元）】；本月取团队员工【目标销售额（元）】汇总值。
  • 日目标 = 月目标 / 月自然天数。
  • 当月剩余目标 = 月目标 - 截止月已完成成交金额。
  • 剩余每日目标 =（月目标 - 截止月已完成成交金额）/ 当月剩余天数。

指标口径

  • 销售数量：商品数量，剔除退货。
  • 购物人数：按证件号 > 手机号 > 小票号优先级取唯一值。
  • 成交单数：不做去重，直接取小票数。
  • 进店客流：北京机场门店（首都机场+大兴机场）通过接口获取。
  • 转化率 = 购物人数 / 进店客流。
  • 连带率 =（销售商品数量 - 退货商品数量）/ 成交单数。
  • 客单价 = 成交金额 / 购物人数。
  • 退货金额、退货件数：取自退货订单，支持查看退货明细；其他值不支持。
